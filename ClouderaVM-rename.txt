Renaming Cloudera VM NAME 

1) Rename the hostname of the Cloudera machine to custom name 
from quickstart.cloudera  to BSGMasterNode.bigstudygroup.com
# hostname BSGMasterNode.bigstudygroup.com

2) Update the /etc/resolv.conf file with the domain name. 

# vi /etc/resolve.conf 

and enter  BSGMasterNode.bigstudygroup.com  in place of Cloudera

3) setup static ip and restart netowrk. 

4) Back up the agent default config file and apply the changes.
